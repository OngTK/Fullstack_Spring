주제
연락처 와 인원수를 처리하는 대기번호 JSP웹페이지 구축

기능
1. 대기 번호 등록 페이지/기능
   조건 : 대기번호는 자동부여 되며 연락처 와 인원수만 입력받습니다.

2. 대기 현황 전체 조회 페이지/기능
   조건 : 대기번호,연락처,인원수,등록일 모든 정보를 전체 출력합니다.

기술스택
JAVA  , DATABASE , *SPRING* , *JSP*
